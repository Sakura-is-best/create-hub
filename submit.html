投稿指南

投稿流程

  1. 在 GitHub 上 Fork 本刊仓库。
  2. 在 content/publications/ 目录下，创建你的论文文件。
  3. 按照 格式规范 撰写你的 Markdown 论文。
  4. 提交 Pull Request。
  5. 编委会审稿通过后合并发表。

投稿要求

  • 内容需与机械动力模组相关。
  • 必须包含摘要、引言、方法、结论等学术要素。
  • 鼓励图文并茂，可附带视频链接。
